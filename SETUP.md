# 🌅 تشيك ليست تيا — النسخة السحابية (مزامنة بين الأجهزة)

هذي النسخة تخليك تشوف من جوالك مباشرة وش شيّك عليه الموظف على جواله — لحظة بلحظة.

تحتاج تسوي شيئين مرة وحدة فقط:
1. تجهيز قاعدة بيانات Supabase (مجانية)
2. رفع الموقع على Vercel (مجاني)

خذها خطوة بخطوة وما راح تاخذ منك أكثر من 15 دقيقة.

---

# 📦 الجزء الأول: تجهيز قاعدة البيانات (Supabase)

### خطوة 1 — إنشاء حساب
- روح على: **supabase.com**
- اضغط "Start your project" وسجّل دخول بحساب Google

### خطوة 2 — إنشاء مشروع جديد
- اضغط "New Project"
- اكتب اسم: مثلاً `teaya`
- اكتب كلمة مرور قوية لقاعدة البيانات (احفظها عندك)
- اختر المنطقة الأقرب (مثلاً Frankfurt أو Bahrain لو متوفر)
- اضغط "Create new project" وانتظر دقيقتين

### خطوة 3 — إنشاء الجدول
- من القائمة اليسار اضغط على **SQL Editor**
- اضغط "New query"
- انسخ والصق الكود التالي بالكامل ثم اضغط "Run":

```sql
create table checklists (
  day_id text primary key,
  staff_checked jsonb default '{}',
  manager_status jsonb default '{}',
  updated_at timestamptz default now()
);

alter table checklists enable row level security;

create policy "allow all" on checklists
  for all using (true) with check (true);

alter publication supabase_realtime add table checklists;
```

- إذا طلع "Success" يعني تمام ✅

### خطوة 4 — نسخ معلومات الاتصال
- من القائمة اليسار اضغط على **Project Settings** (الترس ⚙️)
- اضغط **API**
- بتلقى شيئين مهمين، انسخهم:
  - **Project URL** — مثل `https://abcdefgh.supabase.co`
  - **anon public** key — مفتاح طويل جداً

### خطوة 5 — وضع المعلومات في الموقع
- افتح ملف: `src/supabaseClient.js`
- بدّل السطرين:

```js
const SUPABASE_URL = "ضع_الرابط_هنا";        // ← حط Project URL هنا
const SUPABASE_ANON_KEY = "ضع_المفتاح_هنا";  // ← حط مفتاح anon public هنا
```

- احفظ الملف

# 🚀 الجزء الثاني: رفع الموقع (Vercel)

### خطوة 1 — حساب Vercel
- روح على: **vercel.com**
- سجّل دخول بحساب Google (مجاناً)

### خطوة 2 — رفع المشروع
- اضغط "Add New..." ← "Project"
- ارفع مجلد المشروع كامل (بعد ما حطيت معلومات Supabase فيه)
- اضغط "Deploy"
- انتظر دقيقة

### خطوة 3 — الرابط الدائم
- بيعطيك رابط مثل: **`teaya-morning.vercel.app`**
- هذا الرابط دائم ويشتغل 24 ساعة ✅

---

# 📱 كيف تستخدمه

- **وزّع الرابط على الموظفين** عبر الواتس
- الموظف يفتح الرابط ويضغط على المهام لما يخلصها
- **أنت من جوالك** تفتح نفس الرابط ← تضغط تبويب "المدير"
- تشوف لحظة بلحظة وش شيّك عليه الموظف، وتضغط ✅ Pass أو ❌ Fail
- كل التغييرات تظهر فوراً على كل الأجهزة 🟢

### نصيحة: احفظه كأيقونة على الجوال
- افتح الرابط في المتصفح
- اضغط "إضافة إلى الشاشة الرئيسية"
- بيصير مثل التطبيق على جوالك

- **🔴 أحمر دائماً**: راجع أن `supabaseClient.js` فيه الرابط والمفتاح الصحيحين
- **ما يحفظ**: تأكد أنك شغّلت كود SQL في خطوة 3 كامل
- **المزامنة ما تشتغل**: تأكد من آخر سطر في كود SQL (`add table checklists`)
